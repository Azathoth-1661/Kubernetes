### 傳統部屬時代
各個組織機構在物理服務器上運行應用程序。   
無法為物理服務器中的應用程序定義資源邊界，這會導致資源分配問題。   
例如，如果在物理服務器上運行多個應用程序，則可能會出現一個應用程序占用大部分資源的情況， 結果可能導致其他應用程序的性能下降。  
一種解決方案是在不同的物理服務器上運行每個應用程序，但是由於資源利用不足而無法擴展， 並且維護許多物理服務器的成本很高。
